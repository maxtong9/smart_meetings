# How to run Docker

## Docker compose

`docker-compose up`

To run one service:

`docker-compose up <service>`

To run in the background, use the "detached" flag:

`docker-compose up -d`

To stop services in detached mode:

`docker-compose stop`

To bring everything down and remove containers completely:

`docker-compose down`
